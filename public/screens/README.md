# ZecruAI Website Screenshots

Drop real desktop screenshots into this folder before launch. Do not use mockups or fabricated UI.

Required files:

- `canvas.png` - the mission canvas or multi-agent workspace view.
- `agent-chat.png` - an active planner/coder chat showing a real agent session.
- `project-memory.png` - project memory or mission distillation UI.

Recommended capture notes:

- Use the current production desktop build.
- Hide secrets, local tokens, API keys, and private project paths.
- Keep images at 1600px wide or larger when possible.
